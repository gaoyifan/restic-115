# 概述

**<font style="color:rgb(26, 37, 53);">一、“115生活”简介</font>**

<font style="color:rgb(26, 37, 53);">“115生活”是一款面向个人用户的数字生活平台，提供海量数据的安全存储，多端同步与快速访问。用户不仅可以便捷地管理和使用各类数字资源，还能尽享多维社交、生活服务等多元化功能，畅享高品质的数字生活。</font>

**<font style="color:rgb(26, 37, 53);">二、115生活开放平台能力说明</font>**

<font style="color:rgb(26, 37, 53);">115生活开放平台功能提供“115生活”数据存储、同步、管理等功能的API服务，开发者通过对接 API 的方式，可以方便快捷地将“115生活”云存储能力集成到开发者的应用中。</font>

<font style="color:rgb(26, 37, 53);">目前，已开放的能力包括：</font>

**<font style="color:rgb(26, 37, 53);">•用户管理能力：</font>**<font style="color:rgb(26, 37, 53);">用户授权与信息查询等；</font>

**<font style="color:rgb(26, 37, 53);">•文件管理能力：</font>**<font style="color:rgb(26, 37, 53);">获取文件列表，查看文件详情，文件上传、下载、搜索、移动、删除等；</font>

**<font style="color:rgb(26, 37, 53);">•视频管理能力：</font>**<font style="color:rgb(26, 37, 53);">视频文件的在线转码与播放等；</font>

**<font style="color:rgb(26, 37, 53);">•云下载服务：</font>**<font style="color:rgb(26, 37, 53);">获取云下载任务列表、配额信息，下载任务添加、删除等；</font>

**<font style="color:rgb(26, 37, 53);">•商业价值转化：</font>**<font style="color:rgb(26, 37, 53);">推出“推广产品得收益”计划，开发者可基于用户实际购买的产品获取相应推广收益。</font>
